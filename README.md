# 2D Platformer Demo

This is a hobby project I made in 2025. It is a demo-only Unity 2D platformer project meant for experimentation and learning.

## Overview

A small Unity 2D platformer demo showcasing basic player movement, collectibles, and platform mechanics. The project was created as a personal hobby project and is not intended to be a full commercial game.

## Features

- 2D platforming movement
- Collectible items
- Simple level demo scenes
- Unity Input System support
- Demo-only project structure and assets

## Notes

- Created in 2025 as a personal hobby project.
- Intended as a demo, not a finished product.
- A zipped `.exe` demo build is also available via the GitHub releases page.
- Use it for learning, exploring the setup, or extending the mechanics.

## Project Structure

- `Assets/` - Unity assets, scenes, scripts, and graphics
- `Packages/` - Unity package manifest and dependencies
- `ProjectSettings/` - Unity project settings

## How to Open

1. Open Unity Hub.
2. Add the project folder `2D_Platformer_Demo` if needed.
3. Open `2D_Platformer_Demo.sln` or the project directly in Unity.
